  • 远程数据调试 (扩展代理功能)

    代理数据彻底解决跨域联调等问题 agent配置

    •  
    • 通过正则解决请求转发匹配问题
    •  
    • 通过cookie设置解决SSO用户验证的问题
    •  
    • 支持对本地路径进行方法转换到远程路径
        agent : {
        get:function(path){
            for (var i = 0; i < this.map.length; i++) {
                if( this.map[i].reg.test(path) ){
                    return this.map[i]
                } 
            }
        },
        map:[
            {
                reg : /static/,
                host: 'localhost',
                port: 2850,
                path: function(url){
                    return url.path;
                }
            },
            {
                reg:/xuan/, //路径中若含有xuan字段且映射不到内容，获取远程数据
                host:'xuan.news.cn',
                path:function(url){
                    return '';
                }
            }
        ]
    }
